Installation Instructions

1. Clone the Repository:

Open your terminal or command prompt and navigate to the directory where you want to install the Minesweeper game. Then, run the following command to clone the GitLab repository: git clone https://gitlab.com/your-username/T2-group-1-game.git Replace "your-username" with your actual GitLab username.

2. Navigate to the Game Directory:

Change your current directory to the Minesweeper game folder: cd T2-group-1-game

3. Compile and Run the Java Code:

You can compile and run it directly using the following commands: javac Minesweeper.java java Main

4. Play the ASCII Minesweeper Game:

Once you've executed the commands, the ASCII Minesweeper game should be displayed in the console. Follow the on-screen instructions to play the game. Typically, you will use keyboard controls to uncover tiles and avoid mines.

5. Enjoy the Game:

Play the Minesweeper game and enjoy the ASCII-based experience! If there are specific controls or features, the game's console interface will guide you through them.

Check the README file in the repository for any additional instructions or dependencies specific to the ASCII Minesweeper game. If you encounter any issues during installation or execution, refer to the project documentation or seek help from the game's developers
